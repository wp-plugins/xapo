== Description ==

A WordPress plugin that lets you accept tips on your blog posts.

== Installation ==

1. Upload `xapo-wordpress` to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Enter your API key and secret in the plugin's Settings screen.

== Screenshots ==

1. The plugin adds a Tip button at the end of each post
